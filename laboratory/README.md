# 实验室愿景

## 人机协作的评审系统

### 理想状态

未来的评审系统中，AI 和人类各司其职：

**AI 承担全部客观检查**——逻辑断裂、事实矛盾、时间线错乱、格式规范。这些工作在理想状态下完全自动化，创作者提交稿件后即时获得结构层面的反馈，无需等待人类评审。

**人类聚焦意义导航**——情感冲击力、原创性、角色共鸣、多层解读空间。这些判断依赖生命体验和审美经验，由经过信誉验证的人类评审者提供。AI 不冒充读者，不输出主观评分。

**从规范审美转向描述审美**——系统不输出单一的"好/坏"判断，而是回答"什么样的读者在什么条件下会觉得好"。评价结果是分布式的："85% 的普通读者认为这里有逻辑断裂，但 40% 的高文学素养读者认为这是有意留白"。单一评分不是终点，读者群体画像才是。

**AI 作为模拟镜像**——在作者自检场景中，AI 可以扮演特定读者视角（悬疑爱好者、语言学家、类型编辑），但输出永远标注为"模拟视角"，不参与最终评级。

这不是一个遥远的愿景。L1/L2/L3 分类体系已经划清了边界，剩下的工作是逐步将 L1 任务的精度验证到可商用水平。

### 关键标志

这个愿景实现时，系统会呈现以下特征：

- 创作者收到的评审报告中，L1 项直接标注通过/未通过，无需人工确认
- L2 项标记为"AI 建议"并附置信度，创作者一键确认或驳回
- L3 项不显示 AI 猜测，只显示"等待人类评审"
- 人类评审者的贡献被显性标注和激励，形成可持续的信誉生态
- 评审标准从创作者个人偏好开始，通过使用逐渐演化为社区共识

### 与当前状态的距离

当前已有 L1/L2/L3 分类草案和初步的评级验证数据（来自职场言情系列的 14 篇评审），但距离愿景还有：

- L1 任务的精度尚未经过系统验证（需要人机一致率实验）
- 反馈代币和信誉系统尚未设计原型
- 人类评审的激励机制仅停留在概念层面
- 社区章程的演化机制尚未建模
